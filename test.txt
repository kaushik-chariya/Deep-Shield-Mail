SOP-04 | src/components/data_transformation.py  ⭐ SABSE BADA
----------------------------------------------------------------

FILE: src/components/data_transformation.py
PRIORITY: 4 (Sabse Important Change)
REASON: NLP pehle nahi tha — ab yahan poora NLP pipeline likhna hai

KYA KARNA HAI STEP BY STEP:

STEP 1 — Text Cleaning Function banao:
  - Lowercase conversion
  - HTML tags remove (BeautifulSoup ya regex)
  - URLs remove
  - Punctuation remove
  - Numbers remove (optional)
  - Extra whitespace remove
  - Stop words remove (optional — TF-IDF khud handle karta hai)

STEP 2 — Vectorizer choose karo:

  OPTION A — TF-IDF (Recommended for spam):
    from sklearn.feature_extraction.text import TfidfVectorizer
    vectorizer = TfidfVectorizer(
        max_features=50000,
        ngram_range=(1,2),
        sublinear_tf=True,
        analyzer='word',
        stop_words='english'
    )

  OPTION B — CountVectorizer (simple bag of words):
    from sklearn.feature_extraction.text import CountVectorizer

  OPTION C — Word Embeddings / BERT:
    (alag SOP chahiye hoga — batao to likhta hoon)

STEP 3 — Fit only on TRAIN data:
  X_train = vectorizer.fit_transform(train_text)
  X_test  = vectorizer.transform(test_text)      # ← SIRF transform, fit nahi!

STEP 4 — preprocessing.pkl save karo:
  - Vectorizer ko pickle mein save karo
  - Agar alag text cleaner bhi hai toh use bhi save karo
  - Path: artifacts/<timestamp>/data_transformation/transformed_object/preprocessing.pkl

STEP 5 — .npy files save karo:
  - train.npy — transformed train features
  - test.npy  — transformed test features
  - Path: artifacts/<timestamp>/data_transformation/transformed/

COMMON MISTAKES:
  ❌ Test data pe fit_transform mat karna — sirf transform
  ❌ Vectorizer save karna mat bhoolna — prediction mein chahiye
  ❌ Label (y) ko transform mat karna — sirf text features transform honge
